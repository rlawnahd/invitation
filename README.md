## 모바일 청첩장 사이드 프로젝트

1.  기술스택

    -   Next.js App Router
    -   Typescript

2.  라이브러리

    -   Tailwind CSS (css)
    -   next-themes (next.js 다크모드)

3.  데이터베이스

    -   MongoDB

4.  배포

    -   Vercel

## Getting Started

First, run the development server:

```bash
pnpm dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
```
